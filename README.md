# SAP MDG 客商数据处理工具

处理SAP MDG客商扩充公司代码数据的自动化工具，支持客户/供应商数据去重、排除、排序和批量拆分。

## 功能特性

- **多公司混排**: 支持多个公司数据文件一起排序、一起生成小表
- **文件名解析**: 从文件名自动提取公司代码和事业群（格式：`4290-B.xlsx`）
- **智能排序**: 按分组(Z012>Z002>Z001) > 客商编号 > 公司代码排序
- **按公司分表**: 手工处理清单按公司分表输出
- **保留格式**: 完整保留Excel模板格式
- **自动清理**: 自动删除空行防止SAP报错

## 使用方法

1. 准备数据文件，文件名格式：`公司代码-事业群.xlsx`（如 `4290-B.xlsx`）
2. 运行程序，选择多个数据文件（可多选）
3. 选择模板文件和排除文件
4. 点击"开始处理"

## 输出文件

- `完整模板.xlsx` - 所有公司混排的大表
- `4290需要手工处理的客商.xlsx` - 各公司手工处理清单
- `01.xlsx`, `02.xlsx`... - 混排拆分小表
